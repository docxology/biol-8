# BIOL-1 Syllabus

## Overview

This directory contains the course syllabus for BIOL-1 (Biology 1 at Pelican Bay Prison, Spring 2026). The syllabus is available in multiple formats to accommodate different access needs.

## Syllabus File

- **[BIOL-1_Spring-2026_Syllabus.md](BIOL-1_Spring-2026_Syllabus.md)**: Main syllabus document in Markdown format

## Available Formats

The syllabus is processed to multiple formats for accessibility (all files in flat `output/` directory):

- **PDF**: `output/BIOL-1_Spring-2026_Syllabus.pdf` - Printable document format
- **MP3**: `output/BIOL-1_Spring-2026_Syllabus.mp3` - Audio format for listening
- **DOCX**: `output/BIOL-1_Spring-2026_Syllabus.docx` - Microsoft Word format
- **HTML**: `output/BIOL-1_Spring-2026_Syllabus.html` - Web format
- **TXT**: `output/BIOL-1_Spring-2026_Syllabus.txt` - Plain text format

## Processing

The syllabus is processed using the same automated pipeline as module materials, generating all export formats for maximum accessibility.
