DNA Replication
Build the complementary strand using base pairing rules: A ↔ T, C ↔ G
🔄 Build the Complementary Strand
Base Pairing Rules: Adenine (A) pairs with Thymine (T), Cytosine (C) pairs with Guanine (G). Read the template 3′→5′ and build the new strand 5′→3′.

Template Strand (3′ → 5′):

TAC AAG TTT GCA CCG ATT

Transcription
Convert the DNA template into mRNA — remember: RNA uses Uracil (U) instead of Thymine (T)
📝 Transcribe DNA → mRNA
Key Difference: In RNA, Uracil (U) replaces Thymine (T). So A on DNA → U on mRNA. The other pairs stay the same: T→A, C→G, G→C.

DNA Template Strand:

📚 Mutation Types Summary
🟢
Silent
DNA changes but amino acid stays the same. No effect on protein.
🟡
Missense
One amino acid substituted for another. May alter protein function.
🔴
Nonsense
Creates a premature STOP codon. Truncates the protein.
