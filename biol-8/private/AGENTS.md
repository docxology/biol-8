# BIOL-8 Private Materials Technical Documentation

## Directory Structure

```
private/
├── tests-and-answers/        # Assessment materials with answer keys
├── accessibility/             # Student accommodation documentation
├── other/                    # Additional private materials
├── README.md                 # Overview
└── AGENTS.md                 # This file
```

## File Organization

### Tests and Answers

- **Location**: `tests-and-answers/` subdirectory
- **Content**: Test files, answer keys, grading rubrics
- **Naming**: `test-module-[N].md`, `answers-module-[N].md`, `rubric-module-[N].md`

### Accessibility Accommodations

- **Location**: `accessibility/` subdirectory
- **Content**: Student accommodation documentation, approved accommodations, implementation notes
- **Naming**: `accommodations-[student-id].md` or `accommodations-general.md`

### Other Materials

- **Location**: `other/` subdirectory
- **Content**: Additional private resources, administrative documents
- **Naming**: Descriptive, organized by purpose

## Security and Access Control

### Access Restrictions

- Private materials are never uploaded to Canvas
- Not shared with students or public
- Access limited to authorized instructors
- Proper review before any public sharing

### File Management

- Sensitive information should be properly secured
- Use descriptive but non-identifying names where appropriate
- Maintain clear organization for easy access
- Document access patterns and usage

## Workflow Processes

### Adding Test Materials

1. Create test file in `tests-and-answers/`
2. Create corresponding answer key
3. Add grading rubric if applicable
4. Update documentation as needed

### Managing Accommodations

1. Store accommodation documentation in `accessibility/`
2. Use appropriate naming to protect student privacy
3. Maintain current accommodation records
4. Document implementation procedures

### Material Updates

1. Maintain version control for test materials
2. Update accommodation records as needed
3. Document changes appropriately
4. Ensure security practices are followed

## Quality Assurance

### Pre-Share Checklist

- [ ] Verify material is appropriate for private folder
- [ ] Check for any sensitive information
- [ ] Ensure proper file organization
- [ ] Validate naming conventions
- [ ] Confirm access restrictions are appropriate

### Security Validation

- Regular review of access controls
- Verification that private materials are not in public directories
- Check for accidental exposure of sensitive information
- Validate that Canvas uploads exclude private materials
