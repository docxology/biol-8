# BIOL-8 Resources Technical Documentation

## Directory Structure

```
resources/
├── reference/              # Reference materials (textbooks, papers)
├── media/                 # Videos, animations, interactive content
├── links/                 # External links and curated resources
├── templates/             # Document templates and formatting guides
├── README.md             # Overview
└── AGENTS.md             # This file
```

## File Organization

### Reference Materials

- **Location**: `reference/` subdirectory
- **Content**: Textbooks, scientific papers, reference documents
- **Naming**: Descriptive names with topic or source identifier

### Media Resources

- **Location**: `media/` subdirectory
- **Content**: Videos, animations, interactive content, audio files
- **Naming**: Descriptive names with media type indicator (e.g., `video-`, `animation-`)

### External Links

- **Location**: `links/` subdirectory
- **Content**: Curated links to external websites, databases, educational resources
- **Naming**: `links-[topic].md` or organized by category

### Templates

- **Location**: `templates/` subdirectory
- **Content**: Document templates, formatting guides, reusable materials
- **Naming**: `template-[purpose].md` or descriptive names

## File Naming Conventions

- Use kebab-case for all file and directory names
- Include descriptive prefixes where helpful (e.g., `video-`, `paper-`, `template-`)
- Organize by topic or purpose within subdirectories
- Maintain clear, descriptive names

## Workflow Processes

### Adding Reference Materials

1. Place materials in appropriate subdirectory
2. Use descriptive naming convention
3. Update documentation if needed
4. Ensure proper attribution and licensing

### Managing Media Resources

1. Store media files in `media/` subdirectory
2. Organize by topic or module if applicable
3. Include metadata or description files for complex resources
4. Verify file formats are accessible

### Curating External Links

1. Store link collections in `links/` subdirectory
2. Organize by topic or category
3. Include brief descriptions for each link
4. Verify links remain current

### Creating Templates

1. Place templates in `templates/` subdirectory
2. Include usage instructions in template or separate documentation
3. Use clear naming to indicate template purpose
4. Maintain version control for template updates

## Quality Assurance

### Resource Validation

- [ ] Verify resource is appropriate for course level
- [ ] Check for proper attribution and licensing
- [ ] Ensure file organization follows conventions
- [ ] Validate naming conventions
- [ ] Confirm accessibility of media resources

### Maintenance

- Regular review of external links for currency
- Update reference materials as needed
- Maintain organized structure
